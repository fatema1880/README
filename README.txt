BHROMON BILASH(Tourist Guide App) installation process for Android Server:


1. Click on Google Play Store from any Android Devices.
2. search for Bhromon Bilash App.
3. Click on the app to continue.
4. Using the instruction, make the appropiate selection and entries in each dialog and select Next.
5. Accept the license agreement.
6. Click the install option.
7. After installation, click the Open button to use the app and continue.
